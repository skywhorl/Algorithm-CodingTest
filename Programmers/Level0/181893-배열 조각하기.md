# 배열 조각하기
### [Lv.0 - 181893](https://school.programmers.co.kr/learn/courses/30/lessons/181893)

## 문제
정수 배열 `arr`와 `query`가 주어집니다.
`query`를 순회하면서 다음 작업을 반복합니다.
- 짝수 인덱스에서는 `arr`에서 `query[i]`번 인덱스를 제외하고 배열의 `query[i]`번 인덱스 뒷부분을 잘라서 버립니다.
- 홀수 인덱스에서는 `arr`에서 `query[i]`번 인덱스는 제외하고 배열의 `query[i]`번 인덱스 앞부분을 잘라서 버립니다.

위 작업을 마친 후 남은 `arr`의 부분 배열을 return 하는 solution 함수를 완성해 주세요.
## 풀이
- 인덱스가 짝/홀 기준으로 상황을 나누고 다음과 같은 실행을 하면 된다.
- 풀이에서는 잘라버리는 기능을 `List.RemoveAt()`으로 지웠는데, 지워진 후의 List의 범위(List.Count)가 벗어나지 않게 하기 위해 뒤에서부터 지우는 걸로 구현했다. 

``` C#
using System;
using System.Collections.Generic;

public class Solution {
    public int[] solution(int[] arr, int[] query) {
        // RemoveAt() 사용을 위한 복사용 List
        List<int> applyQuery = new List<int>(arr);
        

        for(int i = 0; i < query.Length; i++)
        {
            // query의 인덱스가 짝수일 때
            if(i%2 == 0)
            {
                // 뒤에서 query의 인덱스 값 직전까지 지우기
                for(int j = applyQuery.Count-1; j > query[i] ; j--)
                {
                    applyQuery.RemoveAt(j);
                }
            }
            // query의 인덱스가 홀수일 때
            else
            {
                // 0부터 query의 인덱스 값 직전까지 지우기
                for(int j = 0; j < query[i]; j++)
                {
                    applyQuery.RemoveAt(0);
                }
            }
        }
        
        return applyQuery.ToArray();
    }
}
```
