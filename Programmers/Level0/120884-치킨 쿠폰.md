# 치킨 쿠폰
### [Lv.0 - 120884](https://school.programmers.co.kr/learn/courses/30/lessons/120884)

## 문제
치킨을 시켜먹으면 한 마리당 쿠폰을 한 장 발급합니다. 
쿠폰을 열 장 모으면 치킨을 한 마리 서비스로 받을 수 있고, 서비스 치킨에도 쿠폰이 발급됩니다.
시켜먹은 치킨의 수 `chicken`이 매개변수로 주어질 때 받을 수 있는 최대 서비스 치킨의 수를 return하도록 solution 함수를 완성해주세요.
## 풀이
- `쿠폰의 개수 = 주문한 치킨의 수`이기에 따로 쿠폰의 변수는 만들지 않았다.
- 서비스로 주문하고서 남은 쿠폰들이 10 미만이어야 하기에 반복문으로 10 이상을 설정하고 계속해서 10으로 나눠주게 해야한다.

``` C#
using System;

public class Solution {
    public int solution(int chicken) {
        int service = 0;
        
        // 쿠폰 개수 = 치킨 주문 수
        while(chicken >= 10)
        {
            // 서비스 계산
            service += chicken / 10;
            // 남은 쿠폰(치킨) 정산
            chicken = (chicken / 10) + (chicken % 10);
        }
        
        return service;
    }
}
```
