# 문자열 밀기
### [Lv.0 - 120921](https://school.programmers.co.kr/learn/courses/30/lessons/120921)

## 문제
문자열 "hello"에서 각 문자를 오른쪽으로 한 칸씩 밀고 마지막 문자는 맨 앞으로 이동시키면 "ohell"이 됩니다. 
이것을 문자열을 민다고 정의한다면 문자열 `A`와 `B`가 매개변수로 주어질 때, 
`A`를 밀어서 `B`가 될 수 있다면 밀어야 하는 최소 횟수를 return하고 밀어서 `B`가 될 수 없으면 `-1`을 return 하도록 solution 함수를 완성해보세요.
## 풀이
- 문자열 옮기는 방법은 `맨 뒷자리의 문자 + 처음~나머지-1 길이의 문자열`을 합쳐 오른쪽으로 미는 것처럼 보이게 했다. 
``` C#
using System;

public class Solution {
    public int solution(string A, string B) {
        int answer = 0;
        
        for(int i = 0; i < A.Length; i++)
        {
            // 문자열이 일치 시 작업 중단
            if(A.Equals(B))
            {
                break;
            }
            
            // 맨 뒷자리의 문자 + 나머지 문자 = 오른쪽으로 미는 것처럼 보임
            A = A.Substring(A.Length-1) + A.Substring(0, A.Length-1);
            answer++;
            
            // 마지막까지 갔는데도 break를 못 했다면 A는 B가 될 수 없음
            if(i+1 >= A.Length)
            {
                answer = -1;
            }
        }
        
        return answer;
    }
}
```

## 다른 풀이
- 프로그래머스 다른 풀이들 중 앞뒤 붙이는 방법을 문자열 두 개로 합쳐 특정 문자 위치는 찾는 `IndexOf()` 함수를사용했다.
``` c#
public class Solution {
    public int solution(string A, string B) {
        int answer = (B+B).IndexOf(A);
        return answer;
    }
}
```
