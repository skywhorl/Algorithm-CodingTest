# 유한소수 판별하기
### [Lv.0 - 120878](https://school.programmers.co.kr/learn/courses/30/lessons/120878)

## 문제
- 소수점 아래 숫자가 계속되지 않고 유한개인 소수를 유한소수라고 합니다. 
- 분수를 소수로 고칠 때 유한소수로 나타낼 수 있는 분수인지 판별하려고 합니다.
- 유한소수가 되기 위한 분수의 조건은 다음과 같습니다. : 기약분수로 나타내었을 때, 분모의 소인수가 2와 5만 존재해야 합니다.
- 두 정수 a와 b가 매개변수로 주어질 때, a/b가 유한소수이면 1을, 무한소수라면 2를 return하도록 solution 함수를 완성해주세요.
## 풀이
- 기약분수가 되려면 최대공약수를 구해서 나눠주면 된다.
- 주어진 각각의 2와 5를 나누었을 때 분모가 1이 되었다면 유한소수고, 이 외의 다른 수가 있다면 무한소수가 된다. 

``` C#
using System;

public class Solution {
    public int solution(int a, int b) {
        
        // gcd : 최대 공약수 준말
        // 최대 공약수를 구하는 이유 : 기약분수로 만들기 위해
        int gcd = GetGCD(a,b);
        
        // 기약분수 표현을 위해 공약수로 나누기
        b /= gcd; 

        // 소인수 2로 나누기
        while (b % 2 == 0) {
            b /= 2;
        }
        // 소인수 5로 나누기
        while (b % 5 == 0) {
            b /= 5;
        }

        // 1이 남았다면 유한소수, 아니면 무한소수
        return (b == 1) ? 1 : 2;
        
    }
    
    // 최대 공약수 구하기
    private int GetGCD(int a, int b) {
        // 나머지 계산으로 공약수 찾기
        while (b != 0) 
        {
            int remainder = a % b;
            a = b;
            b = remainder;
        }
        return a;
    }
}
```
