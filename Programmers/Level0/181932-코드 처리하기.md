# 코드 처리하기
### [Lv.0 - 181932](https://school.programmers.co.kr/learn/courses/30/lessons/181932)

## 문제
문자열 `code`가 주어집니다.
`code`를 앞에서부터 읽으면서 만약 문자가 `"1"`이면 `mode`를 바꿉니다. `mode`에 따라 `code`를 읽어가면서 문자열 `ret`을 만들어냅니다.

`mode`는 `0`과 `1`이 있으며, `idx`를 `0` 부터 `code`의 길이 `- 1` 까지 `1`씩 키워나가면서 `code[idx]`의 값에 따라 다음과 같이 행동합니다.

- `mode`가 `0`일 때
  - `code[idx]`가 `"1"`이 아니면 `idx`가 짝수일 때만 `ret`의 맨 뒤에 `code[idx]`를 추가합니다.
  - `code[idx]`가 `"1"`이면 `mode`를 `0`에서 `1`로 바꿉니다.
- `mode`가 `1`일 때
  - `code[idx]`가 `"1"`이 아니면 `idx`가 홀수일 때만 `ret`의 맨 뒤에 `code[idx]`를 추가합니다.
  - `code[idx]`가 `"1"`이면 `mode`를 `1`에서 `0`으로 바꿉니다.
문자열 `code`를 통해 만들어진 문자열 `ret`를 return 하는 solution 함수를 완성해 주세요.

단, 시작할 때 `mode`는 `0`이며, return 하려는 `ret`가 만약 빈 문자열이라면 대신 `"EMPTY"`를 return 합니다.
## 풀이
- 모드는 두 종류이기에 `bool` 타입으로 편하게 지정했다.
- 모드에 맞게 변경된 상태에서는 짝수든 홀수든 문자열을 추가하는 것이기에 `or` 연산자로 행동을 하나로 묶었다.
- 마지막엔 `answer`가 비는 경우도 생각해야 하기에 return 부분에 삼항 연산자로 상황을 구분했다. 

``` C#
using System;
using System.Collections.Generic;

public class Solution {
    public string solution(string code) {
        string answer = "";
        
        // mode 0 : false, mode 1 : true
        bool mode = false;
        
        for(int i = 0; i < code.Length; i++)
        {
            // 모드를 변경해야 할 때 
            if(code[i].Equals('1'))
            {
                // 모드 바꾸기
                mode = !mode;
            }
            else
            {
                // mode와 idx의 조건에 각각 맞게 정답 추가
                if(mode && i%2 == 1 || !mode && i%2 == 0)
                {
                    answer += code[i].ToString();
                }
            }
        }
        
        // 정답이 비면 조건에 맞게 EMPTY로 수정
        return answer.Equals("") ? "EMPTY" : answer;
    }
}
```
