# 등수 매기기
### [Lv.0 - 120882](https://school.programmers.co.kr/learn/courses/30/lessons/120882)

## 문제
영어 점수와 수학 점수의 평균 점수를 기준으로 학생들의 등수를 매기려고 합니다. 
영어 점수와 수학 점수를 담은 2차원 정수 배열 `score`가 주어질 때, 영어 점수와 수학 점수의 평균을 기준으로 매긴 등수를 담은 배열을 return하도록 solution 함수를 완성해주세요.
## 풀이
- 등수를 비교할 때 이중 반복문으로 전부 비교하게 했고 한 학생의 평균을 다른 학생과 비교하고 성적이 낮으면 등수를 올리게 했다.

``` C#
using System;
using System.Collections.Generic;

public class Solution {
    public int[] solution(int[,] score) {
        
        int length = score.GetLength(0);
        List<int> sum = new List<int>();
        
        for(int i = 0; i < length; i++)
        {
            // 평균을 굳이 도출하지 않아도 순위를 매길 수 있기에 합만 도출
            sum.Add((score[i,0] + score[i,1]));
        }
        
        int[] answer = new int[length];
        for(int i = 0; i < length; i++)
        {
            // 등수 확인할 때 등수를 1부터 시작
            int rank = 1;
            for(int j = 0; j < length; j++)
            {
                // 등수 정하기
                if(sum[j] > sum[i])
                {
                    // 차이가 있으면 등수 갱신
                    rank++;
                }
            }
            // 등수 입력
            answer[i] = rank;
        }
        
        return answer;
    }
}
```
